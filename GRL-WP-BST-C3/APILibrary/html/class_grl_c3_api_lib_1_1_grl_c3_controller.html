◆ GetSignalStrengthValue()

SignalStrengthModel GrlC3ApiLib.GrlC3Controller.GetSignalStrengthValue ( )

Instantanious signal strength value. It is mandatory to start the signal capture before calling this method. Also this is calculated based on the digital ping signals. The maximum value of signal strength is based on the TPR coil selection. Maximum value is considered as 255, and the read voltage will be normalized to 255. Example, if you are using TPR-1A the maxuimum voltage is 12.5V. If the received voltage is 4.2V, the signal strength will be reported as (live voltage/max voltage)*256= 86.

Returns
